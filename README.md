## Aplicação - Urna Eletrônica
> Desenvolvimento de uma urna eletrônica, baseada nas urnas de eleição que pode ser usada em qualquer tipo de eleições.
> Eleição para condominio, empresas e grupos em geral.

### Tecnologias:

* HTML5
* CSS3
* JAVASCRIPT
